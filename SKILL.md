# 官網訂單 → 蝦皮訂單格式轉換

## 腳本位置
`~/.config/opencode/skills/sc-to-shopee/sc_to_shopee.py`

## 使用方式
```powershell
python ~/.config/opencode/skills/sc-to-shopee/sc_to_shopee.py <來源訂單.xlsx> <蝦皮格式.xlsx> [輸出檔名.xlsx]
```

### 參數
| 參數 | 說明 |
|------|------|
| `來源訂單.xlsx` | 官網後台匯出的訂單 Excel（必填） |
| `蝦皮格式.xlsx` | 蝦皮匯入格式範本檔（必填） |
| `輸出檔名.xlsx` | 自訂輸出檔名（選填，預設自動產生） |

### 範例
```powershell
python ~/.config/opencode/skills/sc-to-shopee/sc_to_shopee.py 訂單.xlsx 蝦皮格式.xlsx
python ~/.config/opencode/skills/sc-to-shopee/sc_to_shopee.py 訂單.xlsx 蝦皮格式.xlsx 蝦皮訂單_0710.xlsx
```

## 依賴套件
- `openpyxl`（處理 Excel）

若無 openpyxl，請先安裝：
```powershell
pip install openpyxl
```

---

## 處理流程

### Phase 1：訂單清理
1. **刪除完全空白列** — 無訂單編號也無商品資料的列全部移除
2. **子項目補上母訂單資訊** — 同一訂單多項商品時，將母訂單的購買人、收件人、配送方式等 30+ 個欄位填入子項目列
3. **空白補「—」** — 其餘無法推斷的空白欄位填入 `—` 表示無資料
4. **Campaign 推斷** — 從參照位址 Referer 自動推斷 Campaign Source / Medium（google / line / facebook / email…）

## 注意事項
1. **原始檔案會自動備份**（`原檔名_backup.xlsx`）
2. **範本檔的合併儲存格會自動取消**，避免寫入錯誤
3. 蝦皮範本的標題列（第 1 列）須保持不變，資料從第 2 列開始寫入
4. 若同一訂單有多項商品，每項商品各佔一列（與官網匯出格式一致）
