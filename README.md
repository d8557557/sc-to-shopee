# sc-to-shopee 🛒

**官網訂單 → 蝦皮訂單格式 轉換工具**

將官網後台匯出的訂單 Excel（60 欄）清洗整理後，依欄位對應自動填入蝦皮匯入格式範本（55 欄）。

---

## 功能特色

- ✅ **自動清理** — 刪除空白列、子項目補上母訂單資訊、空白欄位補 `—`
- ✅ **智慧推斷** — 從 Referer 自動推斷 Campaign Source / Medium
- ✅ **格式正規化** — 付款方式（銀行轉帳 / Line Pay / 信用卡）、配送方式自動對應
- ✅ **門市店號解析** — 從超商取貨地址自動擷取門市店號
- ✅ **合併儲存格處理** — 自動取消範本中的合併儲存格
- ✅ **安全備份** — 處理前自動備份原始檔案

## 安裝

```bash
# 依賴套件
pip install openpyxl
```

## 使用方式

```bash
python sc_to_shopee.py <來源訂單.xlsx> <蝦皮格式.xlsx> [輸出檔名.xlsx]
```

### 參數說明

| 參數 | 說明 |
|------|------|
| `來源訂單.xlsx` | 官網後台匯出的訂單 Excel（必填） |
| `蝦皮格式.xlsx` | 蝦皮匯入格式範本檔（必填） |
| `輸出檔名.xlsx` | 自訂輸出檔名（選填，預設自動產生） |

### 範例

```bash
python sc_to_shopee.py 訂單_20260706.xlsx 蝦皮格式.xlsx
python sc_to_shopee.py 訂單_20260706.xlsx 蝦皮格式.xlsx 蝦皮訂單_0710.xlsx
```

## 處理流程

### Phase 1：訂單清理

1. **刪除完全空白列** — 無訂單編號也無商品資料的列全部移除
2. **子項目補上母訂單資訊** — 同一訂單多項商品時，將母訂單的購買人、收件人、配送方式等 30+ 個欄位填入子項目列
3. **空白補 `—`** — 其餘無法推斷的空白欄位填入 `—` 表示無資料
4. **Campaign 推斷** — 從參照位址 Referer 自動推斷 Campaign Source / Medium

### Phase 2：格式轉換

| 蝦皮欄位 | 來源欄位 | 備註 |
|---------|---------|------|

## 注意事項

- 原始檔案會自動備份（`原檔名_backup.xlsx`）
- 範本檔的合併儲存格會自動取消
- 蝦皮範本的標題列（第 1 列）須保持不變，資料從第 2 列開始寫入
- 同一訂單多項商品時，每項商品各佔一列
